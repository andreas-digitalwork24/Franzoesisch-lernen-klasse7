Grundwortschatz & Redewendungen

1. Begrüßung, Befinden und Verabschiedung

  • Salut – Hallo / Hi (unter Gleichaltrigen)
  • Bonjour – Guten Tag / Hallo (allgemein oder formeller)
  • Coucou – Kuckuck / Hallo (sehr umgangssprachlich)
  • Salut, les garçons ! – Hallo Jungs!
  • Salut, les filles ! – Hallo Mädchen!
  • Salut les copains ! – Hallo Freunde!
  • Ça va ? – Wie geht's?
  • Ça va. / Moi, ça va ! – Es geht. / Mir geht es gut!
  • Super ! – Super! / Toll!
  • Bof – Naja. / Geht so.
  • À plus ! – Bis dann! / Tschüss!
  • Au revoir – Auf Wiedersehen
  • À demain – Bis morgen

2. Persönliche Informationen (Name, Alter, Herkunft)

  • Tu t'appelles comment ? – Wie heißt du?
  • Je m'appelle… / Moi, c'est… – Ich heiße… / Ich bin…
  • Tu as quel âge ? – Wie alt bist du?
  • J'ai … ans. – Ich bin … Jahre alt.
  • Il / Elle a quel âge ? – Wie alt ist er / sie?
  • Il / Elle a … ans. – Er / Sie ist … Jahre alt.
  • Tu es d'où ? – Woher kommst du?
  • Je suis de… – Ich bin aus… / Ich komme aus…
  • J'habite à… / près de… – Ich wohne in… / in der Nähe von…

Die Zahlen 1–20:
un (1), deux (2), trois (3), quatre (4), cinq (5), six (6), sept (7), huit (8), neuf (9), dix (10), onze (11), douze (12), treize (13), quatorze (14), quinze (15), seize (16), dix-sept (17), dix-huit (18), dix-neuf (19), vingt (20)

3. Familie, Freunde und Personen

  • la famille / les parents – die Familie / die Eltern
  • la mère (la maman) / le père (le papa) – die Mutter (Mama) / der Vater (Papa)
  • les frères et sœurs / le frère / la sœur – die Geschwister / der Bruder / die Schwester
  • le demi-frère / la demi-sœur – der Halbbruder / die Halbschwester
  • les grands-parents / le grand-père / la grand-mère – die Großeltern / der Großvater / die Großmutter
  • l'oncle / la tante / le cousin / la cousine – der Onkel / die Tante / der Cousin / die Cousine
  • l'ami(e) / les amis / le copain / les copains – der Freund (die Freundin) / die Freunde / der Kumpel
  • le garçon / la fille – der Junge / das Mädchen
  • C'est qui ? / Qui est… ? – Wer ist das? / Wer ist…?
  • C'est… / Ce sont… / Il (Elle) s'appelle… – Das ist… / Das sind… / Er (Sie) heißt…
  • Tu as des frères et sœurs ? – Hast du Geschwister?

4. Haustiere

  • l'animal / les animaux – das Tier / die Tiere
  • le chien / le chat / le poisson – der Hund / die Katze / der Fisch
  • le lapin / la tortue / le rat – das Kaninchen / die Schildkröte / die Ratte
  • Tu as un animal ? – Hast du ein Tier?
  • Je voudrais un chien ou un chat. – Ich hätte gerne einen Hund oder eine Katze.

5. Interessen, Vorlieben und Hobbys

  • J'aime… / Tu aimes… ? – Ich mag… / Magst du…?
  • Je n'aime pas… / Je déteste… – Ich mag … nicht. / Ich hasse…
  • J'adore… / Je suis fan de… – Ich liebe… / Ich bin Fan von…
  • Mon truc, c'est… – Mein Ding ist…
  • … ce n'est pas mon truc. – … das ist nicht mein Ding.
  • oui / non / moi aussi / moi non / moi si / moi non plus – ja / nein / ich auch / ich nicht / ich doch / ich auch nicht
  • le sport / le foot / le basket – der Sport / der Fußball / der Basketball
  • l'escalade / les arts martiaux / la danse – das Klettern / der Kampfsport / das Tanzen
  • les jeux vidéo / jouer (à Minecraft) – die Videospiele / (Minecraft) spielen
  • la musique / la guitare / le piano – die Musik / die Gitarre / das Klavier
  • écouter (de la musique) – (Musik) hören
  • regarder (des séries / des vidéos) – (Serien / Videos) schauen
  • le théâtre / le shopping / la nature / dessiner – das Theater / das Shopping / die Natur / zeichnen
  • les mangas / les BD – die Mangas / die Comics
  • mon chanteur préféré (ma chanteuse préférée) – mein Lieblingssänger (meine Lieblingssängerin)
  • mon joueur préféré (ma joueuse préférée) – mein Lieblingsspieler (meine Lieblingsspielerin)

6. Essen, Trinken und Farben

  • la pizza / les spaghettis / les tomates / la salade – die Pizza / die Spaghetti / die Tomaten / der Salat
  • le bleu / le rouge / le vert / le noir / le jaune – Blau / Rot / Grün / Schwarz / Gelb
  • le blanc – Weiß

7. Stadt und Wohnen

  • la ville / le quartier – die Stadt / das Viertel
  • une boulangerie – eine Bäckerei
  • un cinéma / un musée / un magasin – ein Kino / ein Museum / ein Geschäft
  • une gare / un centre commercial – ein Bahnhof / ein Einkaufszentrum
  • un théâtre / un parc / un café / un restaurant – ein Theater / ein Park / ein Café / ein Restaurant
  • une cathédrale / une tour / un stade – eine Kathedrale / ein Turm / ein Stadion
  • l'appartement – die Wohnung
  • l'entrée / le couloir / le salon / la cuisine – der Eingang / der Flur / das Wohnzimmer / die Küche
  • la salle de bains / les toilettes – das Badezimmer / die Toilette
  • la chambre – das (eigene) Zimmer / das Schlafzimmer
  • un lit / une armoire / un bureau / une chaise / une étagère – ein Bett / ein Schrank / ein Schreibtisch / ein Stuhl / ein Regal
  • un ordinateur / une console / des enceintes / un poster / une plante – ein Computer / eine Konsole / Lautsprecher / ein Poster / eine Pflanze
  • C'est mon endroit préféré pour rêver / travailler. – Das ist mein Lieblingsort zum Träumen / Arbeiten.
  • C'est cool / C'est nul / C'est l'horreur / C'est drôle – Das ist cool / Das ist doof / Das ist der Horror / Das ist lustig

  • le collège / la classe / la cinquième – die Mittelschule / die Klasse / die 7. Klasse
  • le prof (la prof) / les élèves – der Lehrer (die Lehrerin) / die Schüler
  • le tableau / la table / la chaise / la porte / la fenêtre – die Tafel / der Tisch / der Stuhl / die Tür / das Fenster
  • le livre / le cahier / la feuille / le stylo / les devoirs / le carnet d'activités – das Buch / das Heft / das Blatt / der Stift / die Hausaufgaben / das Arbeitsheft
  • Écoutez ! – Hört!
  • Regardez ! – Schaut!
  • Lisez ! – Lest!
  • Écrivez ! – Schreibt!
  • Parlez ! – Sprecht!
  • Répétez ! – Wiederholt!

9. Grammatik-Grundwortschatz

  • être / avoir / regarder / écouter / travailler / habiter / jouer / aimer / faire – sein / haben / ansehen / hören / arbeiten / wohnen / spielen / mögen / machen
  • je (j') / tu / il, elle, on / nous / vous / ils, elles – ich / du / er, sie, man / wir / ihr (Sie) / sie (Plural)
  • le, la, l', les – der, die, das, die (bestimmter Artikel)
  • un, une, des – ein, eine, – (unbestimmter Artikel)
  • du, de la, de l', des – verschmolzene Artikel aus de + bestimmtem Artikel
  • mon, ton, son / ma, ta, sa / mes, tes, ses – mein, dein, sein/ihr / meine, deine, seine/ihre / meine (Pl.), deine (Pl.), seine/ihre (Pl.)
  • Ausnahme vor Vokal: Weibliche Nomen, die mit einem Vokal beginnen, erhalten die männliche Form: mon amie / ton amie / son amie (meine/deine/seine Freundin)
  • Il y a… – Es gibt…
  • Où sont les… ? – Wo sind die…?
  • Est-ce que… ? – Frageeinleitung
  • Qu'est-ce que… ? – Was…?
  • Comment / Où / d'où / Quel âge – Wie / Wo / woher / Wie alt
  • ne … pas (n' … pas) – nicht
  • ne … plus (n' … plus) – nicht mehr
  • petit(e) / grand(e) / joli(e) – klein / groß / hübsch
  • sur / sous / dans / devant / derrière / entre – auf / unter / in / vor / hinter / zwischen
  • à côté de / en face de / à droite de / à gauche de – neben / gegenüber / rechts von / links von
